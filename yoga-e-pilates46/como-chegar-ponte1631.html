Como chegar à ponte

 Ou Ponte - exercício ginástico básica que serve como base para realizar acrobacias complexas em acrobacias. Muito bonito quando visto de lado e permite que você para mostrar aos outros o seu nível de aptidão física e flexibilidade.
 Antes de realizar a ponte certifique-se para se aquecer, com foco sobre os músculos e articulações para trás. Posteriormente, quando a ponte será para você um elemento familiar, você pode fazê-lo sem qualquer preparação e demonstrar aos outros. Mas, para iniciantes preliminar "aquecimento" é necessária. Além disso, o sucesso no estudo da ponte devem ser treinados para realizar exercícios preparatórios especiais.

Exercício 1. Deite-se de barriga, braços puxar para cima. Ao mesmo tempo, levante os braços e as pernas, tentando dobrar o máximo possível. Mantenha os joelhos retos. Tente manter esta posição, pelo menos, 1 minuto.

Exercício 2. Fique de joelhos, os pés na largura dos ombros organizar, levante suas mãos. suavemente inclinaram para trás tentando tocar as mãos de sexo. Faça este exercício durante o tempo que o início para começar a tocar o chão com a ponta dos dedos.

Exercício 3: Deite-se de barriga, as mãos, coloque no nível dos quadris e endireitar. Arquear as costas. Em seguida, dobrar os joelhos e levantar a cabeça tocar os dedos dos pés a cabeça.

Exercício 4: Deite-se de costas, dobre as pernas na altura dos joelhos, flexione os braços e colocar mais perto dos ombros, cotovelos apontando para cima. a partir desta posição, endireitando suavemente os braços e as pernas e arqueia para trás, tentar a ponte. Após ter atingido o seu limite, bloquear a posição por alguns segundos.

Saiba como proceder com confiança para fora todos os exercícios preparatórios, incluindo o mais recente - a ponte a partir de uma posição de bruços, avance para o desenvolvimento da ponte de uma posição ereta. Se seus treinos são realizados no salão, ir para as barras de parede. Se você se exercitar em casa, selecione o espaço na parede.

Em pé, de costas para as barras de parede ou uma parede simples, coloque suas mãos, os pés na largura dos ombros arranjar. A partir desta posição inicial inclinado para trás até as mãos até tocar nas paredes ou barras de parede. Continuando a dobrar e mão sobre a mão sobre as barras de parede ou trilhos na parede, cair para a ponte de ginástica. Fixação da ponte por um tempo, voltar à posição inicial, apenas a mão sobre a mão. Faça este exercício para prendê-lo.

A próxima etapa - a saída da ponte, sem o auxílio da parede. Para fazer isso, coloque um tapete de ginástica e, inicialmente, pedir ao técnico ou assistente para segurá-lo. Fica de frente para um técnico ou assistente, organizar as pernas na largura dos ombros, mãos levantar. enquanto o assistente deve começar a segurá-lo para apoiar a parte de trás. Faça inclinar para trás para uma segunda pausa na encosta e, lentamente inferior na ponte. Fixação da posição final, as mãos do impulso e pela encosta, volte para a posição inicial. Suporte de parceiros se recusam somente quando você está pronto para realizar este exercício sozinho.
