Patins: como escolher

 O mau tempo que dura em nossas latitudes até seis meses ou mais, até o verão dá lugar a um ensolarado, claro e quente. O que não é uma razão para passar o tempo ao ar livre? Diversificar o lazer, você pode usar patins em linha, que, aliás, são uma excelente maneira de manter o corpo em boa forma. A principal coisa é não se machucar enquanto equitação. E é preciso ter cuidado para escolher os patins certas.

A maioria das meninas, a escolha de clipes, atribuem grande importância para o design. Entretanto, existem mais importante do que a aparência, características de patins.

O principal critério na escolha de um patim de rodas é estilo de pilotagem preferido. Os patins são divididas entre as que são projetados para patinar agressivo, patinação de velocidade, hóquei no gelo, há o chamado treinamento, bem como para friskeyta, slalom e fitness. Para os iniciantes, com um estilo que ainda não foi decidido, os rolos são recomendados para fitness. Os preços para estes modelos são geralmente baixos.

Patins para a aptidão pode ser sapatos duros ou moles. Sapatos sólidos são feitos de plástico moldado, soft - assemelham-se as sapatilhas superiores altas, mas eles têm uma estrutura rígida para suportar as pernas. Para os iniciantes é melhor escolher rolos com botas macias que são fáceis, confortável e "respirar". Essas desvantagens são menores em comparação com sapatos sólidos, força e uma tendência que se esforçar. By the way, vos lavei botas macias, juntamente com um quadro, removendo apenas as rodas. Estes sapatos devem ser cuidadosamente verificado para apoio lateral. Para fazer isso, colocar em patins, para fixar a alça de clipe e calcanhar, e, em seguida, tentar "encher-se" patins para o lado interno e externo do pé.

Um elemento importante de patins armação é feita de metal ou plástico. Lightweight estrutura metálica ajuda a desenvolver altas velocidades, mas, infelizmente, têm uma depreciação baixa. Molduras de plástico, ao contrário, um bom amortecimento, o pé para que eles não sentir a vibração. Falta de armação de metal é que eles tendem a dobrar, mas não tão sobre o plástico. Os recém-chegados melhor escolher uma armação de plástico. Eles são pesados, não dão disperso, não propenso a deformação, ameaçado apenas arranhões e rachaduras aparecem apenas quando o "esforços máximos."

Separadamente, você pode falar sobre patinação com quadros removíveis, que podem ser alteradas, se necessário. É importante lembrar que a fixação de tais quadros devem ser verificados antes de cada viagem, para evitar lesões devido ao enfraquecimento de parafusos.

Entre os modernos patins, você também pode encontrar modelos com armação de montagem padrão para sapatos. Pense sobre como selecionar vídeos deste recurso, se você pretende tentar sua mão em diferentes estilos de pilotagem.

Escolhendo rolos, preste atenção para a rigidez e diâmetro da roda, indicado por uma barra nas próprias rodas: primeiro diâmetro, seguido de rigidez. Para patinadores inexperientes se ajustar ao diâmetro de 76-82 mm. Diâmetro maior é usado na patinação cross-country e não é recomendado para iniciantes. Infelizmente, quanto menor o diâmetro, a depreciação inferior. Rigidez das rodas para iniciantes recomendo escolher o intervalo 78-80. Alta rigidez adequada para mais patinadores experientes. Entre outras coisas, é importante ter rolamentos montados suas rodas de skate. Quanto maior for a sua classe de precisão, o mais fácil é possível desenvolver uma alta velocidade. Para patins aulas de fitness são mais comumente usados ​​rolamentos com "quinta" e "sétimo" classe de precisão. Rolamentos de roda e patins muito desgaste rapidamente e precisam ser substituídas.

Há recursos adicionais que precisam procurar ao escolher patins. Se você quer montar no conforto, procure um sistema de "anti-choque", o que melhora as características de amortecimento de patins. E para aqueles que valorizam a manobrabilidade e estabilidade, pode ser aconselhável comprar vídeos com o sistema "rokeringa."

Pegue patins, em que você vai ser confortável para andar, é possível apenas através de muitas tentativas, mesmo sabendo adequado às suas especificações. Tamanho deve ser escolhido de modo que os patins firmemente sentado em sua perna. Para este ajuste é necessário para ser realizado em meias que você vai usá-las. Bem, para crianças pode ser comprado patins de deslizamento, o tamanho das quais irão crescer com o pé da criança.

É altamente recomendável para escolher os vídeos só conhecidos, empresas de confiança, e comprá-los em lojas de esportes especializados. Mercado falsificado precárias, como regra, não muito confortável, confiável e aumentar o risco de lesões.
