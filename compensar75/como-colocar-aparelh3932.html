Como colocar aparelho

 Dale Carnegie Outro disse: "Sorria e pessoas a chegar para você!" E o que acontece com aqueles que estão insatisfeitos com seus dentes e, consequentemente, o seu sorriso? Faça belos dentes, você pode usar os suportes de instalação. Depois de algum tempo, a correção de tais sistemas será o seu sorriso de Hollywood.

As cintas são colocadas para corrigir mordida, ou seja, alterar a localização dos dentes. Mas, mais propriamente chamado de "sistema de suporte" o projeto porque a correção é feita não só com a ajuda de chaves.

Depois de cintas - o nome dos pequenos fechos, que são colados aos dentes do paciente utilizando um adesivo especial. A correção está envolvido e uma força especial do arco ortodôntico resistência que faz com que os dentes para se mover e ligaduras que este arco está ligado, e "rezinochki" (tração elástica), pelo qual a mordida é levada à perfeição. Portanto, graças a alguma ação, os dentes estão se movendo para o ortodontista lugar certo e paciente. Às vezes oclusão antes de instalar os sistemas de suporte de exigir a remoção de um ou mais dentes para abrir espaço para a circulação de outros dentes. Esteja preparado para isso.

Antes de instalar o sistema de suporte, é necessário passar as atividades preparatórias. Em primeiro lugar, é necessário fazer uma reorganização completa da cavidade oral, incluindo uma visita ao dentista não é apenas terapeuta, mas periodontista. Todos os dentes têm de ser curada, e periodontista deve ser efectuada a limpeza.

Então você precisa para criar uma imagem panorâmica dos dentes. Esta imagem mostra mais precisamente o ortodontista como é melhor para planear o movimento dos dentes. Se não houver contra, o médico vai para a escolha de chaves directamente para o paciente.

Eles são divididos em tipos de acordo com sua localização (suportes interiores e exteriores). Cintas externas são mais visível, ao contrário dos sistemas internos, uma vez que estão ligados ao lado interior dos dentes. No entanto, saiba que nem sempre é cintas internas pode corrigir mordida, às vezes pode ser feito somente sistema externo.

Depois de selecionar um grampo para o paciente a fazer uma compromissos preliminares, e no dia marcado, o paciente vem para a instalação de chaves. O procedimento para instalar os suportes leva um pouco mais de uma hora, mas o resto do alinhamento dos dentes podem ter até dois anos. Então, estar preparado psicologicamente para isso. Depois de instalar o paciente entre colchetes deve ser regularmente visitar o ortodontista.
