Como para fixar a forma dos pregos

 A maioria das mulheres têm forma não ideal das unhas. Mas eles foram capazes de pegar essas maneiras de criar uma manicure que este defeito é quase imperceptível. Como é que vamos corrigir a forma das unhas?
 Para corrigir o formato das unhas necessário usar ilusões ópticas. Todo mundo sabe que as linhas verticais são desenhadas figura, torná-lo mais magro e horizontal, ao contrário, estão se expandindo. Preto e qualquer tonalidade de escuro também removido alguns quilinhos a mais. Estes mesmos processos podem ser aplicados numa forma ideal de pregos.

Para alongar visualmente as unhas, use brilhantes, vernizes escuros. Muitas vezes as mulheres estão tentando colocar em suas unhas curtas cores brilhantes. A partir desta malmequeres ainda são invisíveis e aumenta apenas visualmente de largura. Não usar padrões de unhas curtas com linhas horizontais. Adicione o padrão vertical no centro da haste, sem trazê-la para a largura da borda. Você não pode sequer aplicar o verniz nas bordas exteriores. Em seguida alongar visualmente o prego significativamente.

Para as unhas curtas é uma jaqueta boa opção. Sua ponta branca vai criar a ilusão de unha regrown. Unhas curtas agradável e original parece jaqueta angular.

Se você é unhas curtas e largas, eles devem ser estreita. Há também uma opção ideal seria a aplicação de um padrão longitudinal, o que também pode ser separado a partir da superfície do lado das linhas verticais. Em seguida, no olho vai atacar o primeiro desenho, e os lados ir para as sombras. Note-se também que as grandes unhas podem não ser pintar completamente o padrão - esta será uma ainda maior. Não aplique uma grande teste padrão, é melhor escolher padrões pequenos. Maravilhosamente vai olhar como desenho feito na diagonal.

Para corrigir a forma irregular da unha, você deve usar uma combinação de cores diferentes. Então, se as unhas são estreitadas para sua borda livre, melhor as dicas cobrir um verniz luz, e para a causa de base mais escura. Se a ponta é muito mais ampla, então é melhor para cobrir o tom escuro e iluminar a base. Neste caso, os padrões verticais são também adequados. Faça linhas paralelas - o que irá desviar a atenção da opinião de uma forma irregular. Se você quiser fazer as unhas mais proeminente - Aplique um padrão brilhante para o centro da placa ungueal.
